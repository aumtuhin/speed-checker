# Welcome to Speed-Checker

## Project info

A modern, real-time internet speed testing application built with React and TypeScript. Test your download speed, upload speed, and network latency with a beautiful, animated interface.


## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Cloudflare Speed Test API - Reliable speed testing infrastructure
